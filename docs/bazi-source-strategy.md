# 사주명리 엔진 — 리서치 소스 국가 전략 보고서 (中·日·韓)

> 목적: 엔진의 명리 이론·명례(命例) 보강을 위한 자료 검색을, 중국·일본·한국 중 어디에 집중할지 결정한다.
> 근거: (1) 엔진 코드/지식모듈 구조 분석, (2) 3국 명리 학술 지형 웹 리서치(중·일·한·영), (3) 핵심 비교 주장에 대한 적대적 팩트체크(17건 중 confirmed 6 / overstated 11 → 과장은 본문에서 교정 반영).

---

## 1. 결론 (TL;DR)

**"중국에 집중하라"는 직관은 절반만 맞다.** 정답은 단일 국가 집중이 아니라 **역할 분리형 3국 계층 전략**이다.

- **이론·원전·명례 물량 → 중국 (1차, primary)**
- **한국어 출력 어투·방법론 검증·학술 필터 → 한국 (대체 불가능한 2차 working layer)**
- **일본 → 좁은 보조** (적천수 계열 現代型·透派, 古典 일본어 역주에 한정)
- 대략 가중치: **中 50~60% / 韓 30~40% / 日 10% 미만**

그리고 **"검색 자료의 질이 떨어진다"는 체감의 절반은 국가 문제가 아니라 채널 문제다** — 세 나라 모두 오픈웹(검색 첫 페이지)은 算命/철학관/점술앱 SEO 스팸이 압도한다. **채널 고정(화이트리스트 도메인)** 만으로도 품질이 재현적으로 올라간다(§6).

## 2. 진단 — 왜 "동일하게 검색했는데 질이 낮게" 느껴졌나

| 원인 | 설명 | 처방 |
|---|---|---|
| **채널 미고정** | 일반 검색은 3국 모두 상업 노이즈가 신호를 덮는다. 중국은 算命 SEO, 한국은 철학관·작명, 일본은 점술 앱·연예인 listicle. | 국가별 화이트리스트 도메인으로만 검색(§6). |
| **국가 동일 가중** | 세 나라를 같은 비중으로 보면, 원전이 없는 일본 발산파·한국 신살 대중층까지 동급으로 섞여 평균 품질이 내려간다. | 역할 분리(이론=中, 출력=韓, 보조=日). |
| **레이어 혼동** | "원전/주석/명례/통변" 4개 레이어를 한 검색으로 처리하면 잡음↑. | 레이어별로 다른 소스·다른 국가에 매핑(§5). |
| **호환성 미고려** | 盲派(중)·泰山流/算命学(일)처럼 엔진의 용신·격국 로직과 **충돌하는 학파**가 섞이면 "틀린 듯한" 자료로 느껴진다. | 비호환 학파 격리(§7). |

## 3. 엔진이 실제로 필요로 하는 소스 (엔진 구조 기반)

엔진은 **결정론 Python 엔진(= 사실, tier1_facts) + 한국어 프롬프트 모듈(= 해석)** 2층 구조다. 간지·강약·합충·대운·신살·매화괘는 코드가 산출하고, 환각을 막기 위해 **해석은 반드시 `reference_tables.md`의 큐레이션 표와 §⑦ 고전 인용에 앵커**된다. 즉 **소스 품질이 곧 해석 품질**이다.

엔진 지식의 뿌리는 **전부 중국 고전(특정 주석 계보)**으로 핀(pin)되어 있다:

| 엔진 모듈/표 | 핀된 원전·판본 | 보강에 필요한 소스 |
|---|---|---|
| 억부·통관·流通 강약 | 滴天髓 → 任鐵樵《闡微》+ 徐樂吾 補註 | **中** 원전 + **韓** 강해 |
| 격국·용신 선정 | 子平真詮 → 徐樂吾 評註 | **中** 원전 + **韓** 완역 |
| 조후(調候用神) 표 §⑦-3 | 窮通寶鑑 → 徐樂吾《造化元鑰》 | **中** 원전 + **韓** 강해 |
| 신살(神煞) §⑩ | 三命通會(萬民英) | **中** 四庫 완본 |
| 매화역수 괘 | 周易 王弼注 | **별도 매화역수 텍스트** (子平 소스로 안 채워짐) |

엔진 분석에서 드러난 **가장 큰 공백 두 가지**가 이번 결정의 핵심이다:

1. **§⑦ 인용 다수가 `needs_curation`** = 사람이 원전과 대조 검증해야 하는 상태. → **검증은 반드시 중국어 named 印本(任注/徐評)으로** 해야 한다. (한국어 강해는 보조 그라운딩.)
2. **명례(case study) 데이터베이스가 아예 없음** ("50~100개 검증 명식 큐레이션"이 최우선 개선과제로 식별됨). → **물량 주공급은 중국, 어투·검증 공급은 한국.**

### 팩트체크로 교정된 과장 (의사결정 시 반드시 반영)
- ❌ "중국 전통이 가장 일관·조직적" → **과장.** 중국 전통은 오히려 **내부 논쟁적**(子平真詮 격국파 vs 滴天髓 강약파). 물량·다층성은 사실이나 *형식적 조직화*는 오히려 **한국(학위·학술지)**이 앞선다.
- ❌ "ctext = canonical ground-truth" → **아님.** ctext는 "가장 편한 무료 全文"일 뿐 편집가능(Wiki/OCR) 영역. 滴天髓는 **闡微 vs 徵義 이본** 차이가 크므로 인용 전 named 印本(중화서국/무릉 徐樂吾본·袁樹珊 校本) 대조 필수. 三命通會 維基판은 卷10~12 결락 → 신살은 **ctext 四庫 완본** 사용.
- ❌ "한국이 프레임워크 적합성에서 고유" → **과장.** 격국/용신 프레임은 중국에서 물려받은 공유 자산. 한국의 *진짜* 차별점은 좁다: **출력 언어 + 인용 가능한 命理 전용 학술 코퍼스 + 통변/상담 어투.**
- ❌ "신살 강조 = 한국 가치" → **오해.** 神煞 강조는 오히려 일본 전통에서 발달, 古典 子平은 신살을 의도적으로 약화.
- ❌ "일본은 균일하게 약함 / 이론 정련에 가장 약함" → **부분 반박.** 일본은 "약함"이 아니라 **혼합 품질**. 現代型(滴天髓 최고텍스트화·신살 배제)·透派·古典 일본어 역주는 오히려 엔진 적천수 스파인과 잘 맞는다.
- ❌ "盲派 = 강약 완전 무시" → **과장.** 盲派도 旺衰를 做功 분석의 substrate로 유지. 폐기하는 것은 *日主 旺衰 기반 用神 선택*뿐. 충돌은 **엔진 용신 선택층에 한정**.
- ✅ confirmed: 5대 원전 전부 중국산 / 한국어 3대 고전 완역 존재 / 채널 화이트리스트가 S/N을 재현적으로 개선.

## 5. 권고 전략 — 엔진 needs별 계층 매핑

### 니즈 A — 해석 로직 정련 (DEEP THEORY) · `§⑦ needs_curation 해소`
**1차: 중국 원전 + 한국어 강해 병행.**
- 억부·통관·流通: ctext《滴天髓闡微》(任注) + 徐樂吾 補註 — **任注 vs 徐註 구분 인용**. 韓: 박주현《적천수강해》, 맹기옥 나이스 적천수.
- 격국·용신: 子平真詮 + 徐樂吾 評註. 韓: 박영창《완역 자평진전》(판본 대조), 이을로 강해.
- 조후: 窮通寶鑑/欄江網 + 徐樂吾 評註(維基文庫 全文). 韓: 맹기옥 나이스 난강망.
- 신살: 三命通會 卷三(ctext 四庫 완본 — 維基 결락 주의).
- 현대 체계화 가교: 陸致極 — 단, *결정론적 수치화 모델 저자가 아님*. "가장 학술적·source-critical한 이론 지도"로만 활용.
- 일본 보조: 滴天髓 일본어 역주(陽史明·武田考玄), 小山眞樹代로 文言文 난해 구절 disambiguation.

### 니즈 B — 명례 (CASE STUDY) · `명식 DB 신설`
**중국=물량 주공급, 한국=어투·검증 공급. 반드시 등급 분리.**
- 논리 패턴/해석 구조 학습: 滴天髓闡微·子平真詮·窮通寶鑑 예시 — **위조 가능성 → '사실 검증' 금지, '추론 패턴'만.**
- 실명 검증용: 韋千里《千里命稿》, 袁樹珊《袁氏命譜》(역사 인물 = 검증 가능 생년).
- 大運/세운 응기 보강: 梁湘潤《子平母法大流年判例》, 鍾義明《現代破譯滴天髓》, 段建業 盲派 案例(**做功 관계기법·象만, 용신 verdict 제외**).
- 한국어 통변 어투 + 검증: 이석영《사주첩경》, 맹기옥/박주현 해설서, 큐레이션 카페(역학동·원제역학), KCI/RISS 논문 내 명식 분석.

### 니즈 C — 한국어 출력 텍스트 (OUTPUT REGISTER)
**한국이 1차.** 통변 phrasing·십신/격국/용신 설명 register·신살/물상 어휘를 맹기옥·박주현·박청화(물상 imagery) 등에서 차용. 용어는 한글(한자 병기) 그대로 — 격국格局·용신用神·억부抑扶·조후調候·통관通關·십신十神·12운성·신살神煞 — 엔진 어휘와 1:1.

## 7. 리스크 및 주의

1. **비호환 학파 격리 (가장 중요).**
   - 盲派(段建業 등): 用神·格局 폐기 → **강약/용신 selector 금지**, 관계기법·象·실례만 별도 라벨 채널로. (旺衰는 substrate로 유지하므로 "盲派=강약 무시"로 과단정 금지.)
   - 일본 泰山流(蔵干 용신·격국/신살/공망 폐기), 算命学(三柱·人体星図·天中殺), 透派 五行点数 강약 → 엔진의 결정론 strength/relations facts에 **무비판 통합 금지**, 대조용으로만.
   - 한국 신살·당사주 대중층 → 적천수 억부 순수 로직에 신살 folk 규칙 혼입, case text 필터 필요.
2. **简体/繁體 + 이본.** 适天髓/適天髓·真诠/真詮 표기 갈림(ctext 토글). **滴天髓 闡微 vs 徵義 채택 이본 명시 + named 印本 대조.**
3. **상업 노이즈.** 채널 고정 없이는 3국 모두 noise 매몰 — §6 화이트리스트가 재현적으로 품질 개선(confirmed).
4. **번역 충실도.** 中→韓: 한자어 직접 차용 → 손실 최소(일본어 경유보다 작음), 단 文言文→현대→한국어 가공 레이어 필요. 日→韓/中: 通変星=십신, 蔵干=지장간, 十二運=12운성, 三柱(算命学)≠四柱 매핑 필수. 민국 주석(徐樂吾)은 **원전(PD)과 달리 주석 apparatus 저작권 잔존 가능.**
5. **명례 진위.** 古典 명례 = 사실 검증 불가, 논리 예시로만. 실명 검증은 韋千里·袁樹珊 命譜 또는 한국 논문 검증 사례.
6. **모듈별 공백.** 매화역수(梅花易數)는 子平과 별개 술수 → 子平 case가 안 채움(한국은 더 얇음). **별도 매화역수 텍스트 확보 필요.**
7. **권위 판정.** peer-review는 (한국 학술층 제외) 부재 → 中·日은 판본학·주석 계보·저자 credential로 대체. 한국 KCI/RISS는 엔진 설계 논쟁(신살 무용론, 격국 vs 억부, 진태양시·동경 135도 표준시)의 근거로 활용.
